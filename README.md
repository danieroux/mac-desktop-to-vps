# This is a collection of goodies to make accessing my macOS Desktop from my OpenClaw VPS easier/smoother 

## inbox-reminders

The simplest proxy of my Inbox List in Apple reminders, so that OpenClaw can get to it and work with it.

Considerations: If `node` goes away because of GC, this will go away.

## AssistantArthurProfile

We are _not_ committing this, but it is the Chrome persistent profile _with_ passwords and Credit Cards - all _not_ the important ones
